Lemonadebox

A Fork of the usual AMPP server... specially designed for DJ's to eliminate the need for people to walk up to request music.

Download this project as a .zip file Download this project as a tar.gz file

Introducing Lemonadebox by Aerobyn.

So you're a DJ. You're doing your DJ stuff. But then you look and see a long line of people waiting to request a song. That can be exasperating. So we've found an alternative.

Fast

Lemonadebox runs Debian linux on an ARM based raspberry pi.

Lightweight

Around the size of a few credit cards, Lemonadebox doesn't take up a lot of room.

Gets the job done

Loaded with Mixxx mp3 dj software, You can do all of your DJ'ing with one nifty little package, not to mention Lemonadebox supports multiple soundboards.

Open source.

You can install the Lemonadebox AMPP stack to your raspberry pi to do your own Lemonadebox DIY project. Software and trademarks belong to their respeced owners.
